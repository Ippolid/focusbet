# Screenshots

Drop captures here with these exact filenames so the main README picks them up:

| File | Screen to capture |
| --- | --- |
| `hero.png` | Wide hero shot — the dashboard or a running focus timer (≈720–1280px wide) |
| `timer.png` | Focus timer with the smooth progress bar |
| `result.png` | The "FOCUS DONE" fork (Continue / Rest / Play) |
| `slots.png` | Slots — ideally a win with confetti |
| `mines.png` | Mines board mid-game |
| `roulette.png` | Roulette wheel / result |
| `stats.png` | Statistics screen |
| `settings.png` | Settings with the custom preset rows |

Tips:
- Use a dark terminal theme and a clear monospace font.
- A 1280px-wide capture looks crisp on GitHub; the hero can be a touch smaller.
- For the games, `FOCUSBET_TIME_SCALE=60 focusbet` gets you to a result fast.
